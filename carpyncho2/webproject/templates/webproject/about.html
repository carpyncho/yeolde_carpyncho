About Carpyncho

We present the IVOA registered VVV band-merged photometry facility for the Virtual Observatory, first published on the 2013-10-08T14:00:00 and a new data mining facility in development which we hope will be used by VVV team members and eventually the public to search and characterize time variable data of the ~ PB size VVV database.

Carpyncho is layored on-top of a scientific Python library stack which includes: NumPy, SciPy, Astropy, PyAstronomy, astroML and scikit-learn.
